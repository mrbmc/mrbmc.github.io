<b>The state of the Art</b>

This site has no popup windows.  
This site has no loading screens.  
This site has no intro animation.  
This site has no splash screen.  
This site isn't a fixed width.  
The content fits on your screen without scrolling.
This site has few heavy graphics SO IT LOADS ALREADY DAMNIT.
This site is NOT under redesign.  Even if it's mediocre, it's up.
This site will use all your CPU power.

Did you notice any of that before i said it?  I didn't think so.  That's ok, because good design, should be transparent, it should facilitate the task at hand.  This site, has no task at hand really, it's just mental masturbation, but at least it loads fast :D  To quote the great Ben Running, "This is site is amazingly full of yet deviod of real content at the same time."

<b>Why is ths site the way it is?</b>

We have a basic html list of toys and blogs instead of a fancy flash interface so as to leave your processor free to munch away on the flash experiments over on the left.  besides... It would be stupid to create a fancy interface to navigate fancy interfaces wouldn't it?

We have a key so maybe you'll know what you're getting into when you click.

What's up with the poll?  Did you vote?  I think that explains it.

We use iFrames because your browser's built in scroller is way better than a DHTML or Flash widget and it kind of uses a <i>lingua franca</i> interface device.  See... that's latin.  Latin is like smart and stuff.
